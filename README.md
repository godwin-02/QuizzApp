# ⚡ RapidSync: Elite Mobile Quiz

**RapidSync** is a high-performance, glassmorphism-styled web quiz application optimized for mobile devices. It features real-time category syncing, a "Lightspeed" bonus system, and a persistent local leaderboard.

## ✨ Key Features

* **Cyber-Glass UI:** A sleek, dark-mode interface with frosted glass effects and neon accents.
* **Dynamic Sector Syncing:** Fetches fresh questions from the OpenTDB API across Science, History, Sports, and more.
* **Lightspeed Bonus:** Rewards rapid cognitive processing with extra XP for answers provided in under 5 seconds.
* **50/50 Lifeline:** Integrated haptic-feedback lifeline to eliminate two incorrect data nodes.
* **Hall of Fame:** A persistent local scoreboard (LocalStorage) that tracks the top 5 highest-scoring sessions.
* **Mobile First:** Built-in support for the Web Share API and vibration haptics for an immersive experience.

---

## 🛠️ Tech Stack

* **HTML5** - Semantic structure.
* **CSS3** - Custom properties, Flexbox layout, and `backdrop-filter` glass effects.
* **JavaScript (ES6+)** - Async/Await API handling, Web Audio API for procedural SFX, and LocalStorage for data persistence.

## 📥 Local Installation

To run **RapidSync** on your own machine:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/godwin-02/QuizzApp.git](https://github.com/godwin-02/QuizzApp.git)
    ```
2.  **Navigate to the directory:**
    ```bash
    cd QuizzApp
    ```
3.  **Launch:**
    Open `index.html` in any modern web browser.

---

## 🎮 Mission Protocol

1.  **Identify:** Enter your **Codename** on the home screen.
2.  **Select:** Choose your **Sector** (Category) and difficulty level.
3.  **Sync:** Click **Initiate Sync** to pull data from the cloud.
4.  **Decode:** Answer 5 nodes as fast as possible to maximize your XP!
5.  **Share:** Use the built-in share button to broadcast your rank to your network.
